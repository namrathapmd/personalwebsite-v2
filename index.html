Projects

ByteDance Revolution

ByteDance Revolution is an interactive dance web application built using Node.js with Google Firebase, React and Posenet. The Posenet model was used from TensorFlow with React hooks on Audio and Webcam components to compare and grade players on pre-set dance routines, calculate and display scores on. The project workflow was managed using GitHub for branch control, merges and tasks.
 

Technologies Used: Javascript, Nodejs, tensorflow, etc

Grace Shopper

A fully functional mock e-commerce site selling varieties of pasta, built in a span of 4 days and deployed on Heroku. Worked closely with React components/Redux store for cart/checkout logic for a smooth user and guest experience. Also, integrated Google OAuth with Passport to enable logins using google accounts.
 

Technologies used: Front-end built with React, Redux, React-Redux, Redux-Thunk, CSS Node.js backend: Express.js, Sequelize ORM, & Postgres rDBMS, Passport and GoogleOAuth.

Stackbot Inc.

Robotics Inc. is a CRUD web app built over the course of one week while attending Fullstack Academy. This was a solo project assigned to students immediately after we had finished learning all the full stack technologies in the FSA curriculum. Tech used includes React, Redux, Express, Sequelize and PostgreSQL. This my first full stack web app built from start to finish. It served me by helping to solidify all of the concepts, at every level of the stack, that I had learned in the prior six weeks or so.
 

Technologies Used: Javascript, Nodejs, React, Redux, Sequelize, PostgreSQL etc

Configurable Verification of RISC processors

This paper presents a configurable verification environment for RISC processors which was presented as my Graduate Paper at the end of my masters. The verification environment is developed in SystemVerilog. The verification environment will validate the performance of the RISC processors with a micro-architectural model developed in SystemC. The system also has an intelligent Instruction Generator that generates random sequences of instructions. All verification system components are configured using a common RISC processor architecture configuration control file.
 

Technologies Used: SystemVerilog, SystemC, Perl etc
